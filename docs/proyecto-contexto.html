¿Qué es R&R Importaciones?

Un sistema de gestión operativa y financiera multi-tenant diseñado específicamente para agencias de importación vehicular en la frontera México-EEUU (Cliente principal: Nuevo Laredo).

El sistema sustituye las hojas de cálculo heredadas por flujos automatizados de: decodificación de vehículos vía API de la NHTSA, cotización con lookup del tabulador fiscal oficial del Anexo 2, control de pedimentos y sus rectificaciones (R1-R3), control de cobros parciales por transferencias/efectivo, y un portal web público de seguimiento para los clientes finales.

Arquitectura & Stack Tecnológico

Frontend Framework Angular 19.2 (Standalone, Signals)
Backend API .NET 10 (Clean Architecture, REST)
Base de datos PostgreSQL 16 (Dapper + EF Core)
UI & Estilos PrimeNG 19 & Tailwind CSS v4
Integraciones core API NHTSA (VIN) & API Banxico (FIX)
Servicio de IA Google Gemini API (Agente Rodri)

Conceptos Clave del Dominio

  • Apodos de Clientes Se priorizan los nombres de uso informal ("KIKO", "DOMINGO") sobre razones sociales formalizadas.
  • VIN Completo & Corto Almacenamiento de 17 caracteres. Búsqueda y visualización por los últimos 6 a 8 (VIN Corto).
  • Gasto Hormiga Cargable Gastos en patio de flete, pensión o copias se pueden transferir al saldo del cliente final.
  • Comprobantes Obligatorios No se puede registrar un pago sin comprobante PDF/Imagen inyectado. Sin excepciones.

Cálculo Fiscal: POST_2017 (Año modelo >= 2017) Nacional
impuestos_total = IGI + DTA + IVA + PREV - IGI = valor_pesos * 10% - DTA = Max(valor_pesos * 0.8%, $202) - PREV = $161 (Prevalidación fija) - IVA = (valor_pesos + IGI + DTA) * 16% - PRV = $0

Base Gravable de IVA: Suma únicamente el valor del vehículo en pesos, el IGI y el DTA. Las prevalidaciones (PREV, PRV) no forman parte de la base según reglas vigentes del SAT.

Cálculo Fiscal: PRE_2016 (Año modelo <= 2016) Histórico/Frontera
impuestos_total = IGI + DTA + PREV + IVA + PRV - IGI = valor_pesos * 50% - DTA = $408 (Derecho de Trámite Aduanero fijo) - PREV = $240 (Prevalidación fija) - PRV = $290 (Contraprestación fija) - IVA = (valor_pesos + IGI + DTA + PREV) * 16%

IGI al 50%: Tasa impositiva elevada aplicable a vehículos anteriores a las reglas de libre comercio del 2017. El DTA, PREV y PRV son de cuotas fijas establecidas.
