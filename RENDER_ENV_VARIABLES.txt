========== NOTAS ==========

1. VAPID - Valores ya generados y listos para copiar
   - Las claves VAPID son globales para la app
   - El Subject debe ser un email válido (usado para contactarte si hay problemas)

2. WhatsApp - Requiere cambios:
   - AccountSid: obtener de https://www.twilio.com/console
   - AuthToken: obtener de https://www.twilio.com/console
   - FromNumber: es el número Twilio assignado (por defecto sandbox: +14155552671)
   - AdminsGroupNumbers: reemplazar con teléfonos reales separados por coma

3. URL Base - Verificar:
   - Si tu Render app tiene custom domain, cambiar a ese domain
   - Sino, Render asigna automático: YOUR_APP_NAME.onrender.com

4. Base de Datos:
   - Reemplazar valores YOUR_* según tu servidor
   - Si es Railway, Vercel Postgres, etc., copiar connection string tal cual

5. puerto HTTP:
   - Render requiere escuchar en puerto 10000 por defecto
   - Cambiar solo si Render lo especifica diferente
